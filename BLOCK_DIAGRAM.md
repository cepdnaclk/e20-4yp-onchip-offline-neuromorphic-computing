## Summary

This block diagram shows:

1. **Hierarchical Architecture**: 3-level routing (Main → SF-8 → SF-4 → Clusters)
2. **Distributed Weight Memory**: 8 resolvers serving 64 clusters
3. **LIF Neuron Pipeline**: Accumulate → Decay → Threshold → Spike
4. **Dual Networks**: Separate spike and initialization networks
5. **Scalable Design**: Easy to expand by adding more clusters/resolvers
6. **Current Limitation**: READ-ONLY weights (inference only, no training)

**Total Capacity**: 2,048 neurons, 4.2 MB weight storage, fully pipelined operation
